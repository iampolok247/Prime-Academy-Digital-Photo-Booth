================================================================================
STEP-BY-STEP: CREATE YOUR FRAME IN PHOTOSHOP
================================================================================

1. CREATE NEW DOCUMENT:
   - File > New
   - Width: 1200px
   - Height: 1600px
   - Resolution: 72 DPI (web)
   - Color Mode: RGB
   - Background: TRANSPARENT (important!)

2. DRAW TOP SECTION (0px to 250px):
   - Create rectangle: 1200 × 250px at position (0, 0)
   - Fill with gradient: #053867 to #253985
   - Add Prime Academy logo (centered or left-aligned)
   - Add text: "Digital Photo Booth 2025"
   - Add decorative borders/patterns

3. LEAVE CENTER AREA TRANSPARENT (250px to 1350px):
   - DO NOT add anything here
   - Keep completely transparent
   - Or add very subtle border (optional, max 2px)
   - This is where the captured photo will show

4. DRAW BOTTOM SECTION (1350px to 1600px):
   - Create rectangle: 1200 × 250px at position (0, 1350)
   - Fill with matching gradient: #053867 to #253985
   - Add text: "#PrimeFutureStartsHere" (48pt, bold, white)
   - Add text: "@Prime Academy, Bangladesh" (32pt, white)
   - Add decorative elements

5. EXPORT:
   - File > Export > Export As
   - Format: PNG
   - Transparency: CHECK THIS BOX ✓
   - Quality: 8-10
   - Save as: frame.png

================================================================================
STEP-BY-STEP: CREATE YOUR FRAME IN CANVA (FREE)
================================================================================

1. CREATE CUSTOM SIZE:
   - Click "Create a design"
   - Click "Custom size"
   - Width: 1200px
   - Height: 1600px
   - Click "Create new design"

2. SET TRANSPARENT BACKGROUND:
   - If you have Canva Pro: Click "Background" > "Transparent"
   - If you have Free: Skip this, use white background

3. ADD TOP SECTION:
   - Add rectangle: 1200 × 250px at top
   - Fill color: #053867 or gradient
   - Add text elements
   - Upload and add your logo

4. KEEP CENTER EMPTY:
   - Position: 250px from top to 1350px from top
   - Leave completely empty
   - DO NOT add any elements here

5. ADD BOTTOM SECTION:
   - Add rectangle: 1200 × 250px at bottom (position 1350px)
   - Fill color: #053867
   - Add hashtag text (large and bold)
   - Add social handle text

6. DOWNLOAD:
   - Click "Share" > "Download"
   - File type: PNG
   - If Pro: Check "Transparent background"
   - Click "Download"

COMMON MISTAKES TO AVOID
================================================================================

❌ DON'T: Put a photo in the center area
❌ DON'T: Add text over the safe area (250-1350px)
❌ DON'T: Use a white/solid background (must be transparent!)
❌ DON'T: Make the frame too detailed (keeps file size small)
❌ DON'T: Cover the face area with graphics

✓ DO: Keep center transparent
✓ DO: Add graphics only to top and bottom 250px sections
✓ DO: Use brand colors
✓ DO: Test with the green guide
✓ DO: Keep file size under 500KB
